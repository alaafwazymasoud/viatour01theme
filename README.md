# viatour01theme

A modern theme built with Tailwind CSS v4.

## Features

- **Tailwind CSS v4** - The latest version of the utility-first CSS framework
- **Simple Build Process** - Easy to use build and watch commands
- **Responsive Design** - Mobile-first responsive design out of the box

## Getting Started

### Installation

Install dependencies:

```bash
npm install
```

### Development

Build the CSS:

```bash
npm run build
```

Watch for changes and rebuild automatically:

```bash
npm run watch
```

### Usage

Open `index.html` in your browser to see the theme in action.

## Project Structure

```
.
├── src/
│   └── styles.css          # Tailwind CSS source file
├── dist/
│   └── styles.css          # Compiled CSS (generated)
├── index.html              # Sample HTML page
└── package.json
```

## Tailwind CSS v4

This project uses Tailwind CSS v4.1.18, which introduces several improvements:

- **CSS-first configuration** - Configuration is now done in CSS using `@import`
- **Improved performance** - Faster builds and better optimization
- **New features** - Enhanced utilities and better developer experience
